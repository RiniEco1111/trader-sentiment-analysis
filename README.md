# Trader Performance vs Market Sentiment

This project analyzes how market sentiment (Fear/Greed Index) impacts trader behavior and performance.

## Key Insights
- Traders perform better during Fear periods
- Smaller trade sizes generate better returns
- Frequent traders outperform infrequent traders

## Files
- trader.ipynb → full analysis
- trader.html → easy-to-view version
- Final_Trader_Report_With_Charts.docx → summary report
